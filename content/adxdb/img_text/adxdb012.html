说明adxdb012.eps的说明

该图有一个主标签“XML schema，emp.xsd”。它显示了两列XML元素。每列中显示两个元素，每个列元素后面有省略号（“...”）。从每列中显示的下部元素到另一列中显示的上部元素有一个箭头。左列中显示的上部元素是<xs：complexType name =“CT1”...>。较低的是<xs：element name =“e2”type =“CT2”/>。右列中显示的上部元素是<xs：complexType name =“CT2”...>。较低的是<xs：element name =“e2”type =“CT1”/>。
