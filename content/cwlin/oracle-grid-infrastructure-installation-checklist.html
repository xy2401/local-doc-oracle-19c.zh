1 Oracle Grid Infrastructure安装核对表

使用清单来规划和执行Oracle Grid Infrastructure（Oracle Clusterware和Oracle自动存储管理）安装。

Oracle建议您使用清单作为安装规划过程的一部分。使用此清单可帮助您确认服务器硬件和配置是否满足此版本的最低要求，并确保您成功安装。

  • Oracle Grid Infrastructure的服务器硬件清单
    查看Oracle Grid Infrastructure安装的服务器硬件要求。
  • Oracle Grid Infrastructure和Oracle RAC的操作系统清单
    查看Oracle Grid Infrastructure安装的操作系统要求清单。
  • Oracle Grid Infrastructure的服务器配置清单
    使用此核对表可以检查Oracle Grid Infrastructure安装的最低服务器配置要求。
  • Oracle Grid Infrastructure的网络清单
    查看此Oracle Grid Infrastructure安装的网络核对表，以确保您具有群集所需的硬件，名称和地址。
  • Oracle Grid Infrastructure的用户环境配置清单
    使用此清单来规划Oracle Grid Infrastructure安装的操作系统用户，组和环境。
  • Oracle Grid Infrastructure的存储核对表
    查看Oracle Grid Infrastructure安装的存储硬件和配置要求清单。
  • Oracle Grid Infrastructure的群集部署清单
    查看用于规划集群部署Oracle Grid Infrastructure安装的核对表。
  • Oracle Grid Infrastructure的安装程序规划核对表
    在启动Oracle Universal Installer之前，请查看用于规划Oracle Grid Infrastructure安装的核对表。
