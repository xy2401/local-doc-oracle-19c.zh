修补Oracle数据库软件主页

补丁

/ RHP-RESTAPI / RHP / workingcopies / {workingCopyId} /数据库

将数据库从源工作副本移动到修补的工作副本。

请求

支持的媒体类型
  • 应用程序/ JSON
路径参数
  • workingCopyId：string
    工作副本的名称。
标题参数
  • 授权：字符串
    Base64编码用户名和发出请求的用户的密码。有关更多信息，请参阅Fleet Patching和Provisioning的身份验证 。
身体 （ ）
根架构：MoveDatabaseBody
输入： object
显示来源
  • abort（可选）：布尔值
    默认值： false
    中止正在进行的“移动”操作。
  • agpath（可选）：字符串
    持久Oracle主路径的读/写路径。
  • aupath（可选）：字符串
    持久Oracle主路径的只读黄金映像路径。
  • auth（可选）：字符串
    使用身份验证插件访问远程节点。
  • authArgs（可选）：对象 authArgs
    允许的其他属性： additionalProperties
    所选身份验证插件的其他属性。
  • cc（可选）：字符串
    除了工作副本的所有者之外，将向其发送电子邮件通知的用户列表。
  • client（可选）：string
    客户端群集名称
  • continueRestart（可选）：布尔值
    默认值： false
    继续在下一批节点上重新启动数据库。
  • dbname（可选）：字符串
    要配置的数据库（DB_UNIQUE_NAME）的名称。
  • disconnect（可选）：boolean
    默认值： false
    在停止或重新定位服务之前断开所有会话。
  • drain_timeout（可选）：整数（int32）
    服务耗尽超时以秒为单位指定。
  • eval（可选）：布尔值
    默认值： false
    无需执行命令即可进行评估。
  • ignorewcpatches（可选）：boolean
    默认值： false
    如果修补的工作副本缺少源路径或工作副本中存在的某些修补程序，则忽略。
  • keepplacement（可选）：布尔值
    默认值： false
    确保管理员管理的Oracle RAC或Oracle RAC One数据库的服务在移动操作之前和之后的相同实例上运行。
  • noDataPatch（可选）：布尔值
    默认值： false
    表示不应对正在移动的数据库执行“datapatch”。
  • noreplay（可选）：布尔值
    默认值： false
    断开连接时禁用会话重播。
  • notify（可选）：boolean
    默认值： false
    发送电子邮件通知
  • oraclebase（可选）：字符串
    用于供应Oracle数据库主目录或Oracle Grid Infrastructure主目录的ORACLE_BASE路径。
  • path（可选）：string
  • revert（可选）：布尔值
    默认值： false
    在批量移动或智能移动的情况下，还原为Oracle主目录或工作副本。
  • root（可选）：布尔值
    默认值： false
    使用root凭据访问远程节点。
  • saf（可选）：字符串
    服务可用性因子，即移动期间服务必须保持运行的实例的最小百分比。
  • schedule（可选）：string
    执行操作的首选时间，采用ISO-8601格式。例如：2016-12-21T19：13：17 + 05。
  • separate（可选）：boolean
    默认值： false
    分别处理批次。在继续下一批之前，移动操作会暂停以供用户干预。
  • skip（可选）：boolean
    默认值： false
    跳过当前批节点并在下一批节点上重新启动数据库。
  • stopoption（可选）：字符串
    允许值： [ "ABORT", "IMMEDIATE", "NORMAL", "TRANSACTIONAL", "TRANSACTIONAL_LOCAL" ]
    数据库的停止选项：ABORT，IMMEDIATE，NORMAL，TRANSACTIONAL，TRANSACTIONAL_LOCAL。
  • sudoPath（可选）：字符串
    sudo二进制文件的位置。
  • sudoUser（可选）：字符串
    以sudo用户名执行超级用户操作。
  • superUserPasswd（可选）：字符串
    超级用户帐户的密码。
  • targetnode（可选）：string
    需要执行操作的节点
  • useractiondata（可选）：string
    要传递给useraction脚本的useractiondata参数的值
