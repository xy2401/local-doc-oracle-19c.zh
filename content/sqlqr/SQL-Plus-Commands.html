SQL * Plus命令

SQL * Plus是一个命令行工具，提供对Oracle RDBMS的访问。 SQL * Plus使您能够：

  • 输入SQL * Plus命令以配置SQL * Plus环境

  • 启动和关闭Oracle数据库

  • 连接到Oracle数据库

  • 输入并执行SQL命令和PL / SQL块

  • 格式化和打印查询结果

SQL * Plus可在多个平台上使用。

表A-1中显示的命令是命令行界面中提供的SQL * Plus命令。并非显示所有命令或命令参数。

表A-1 基本SQL * Plus命令

数据库操作 SQL * Plus命令

登录SQL * Plus

 
SQLPLUS [
  [{username[/password][@connect_identifier] | / }
  [AS {SYSASM|SYSBACKUP|SYSDBA|SYSDG|SYSOPER|SYSKM}]
  [edition=value]]
        | /NOLOG
        ]

列出SQL * Plus中提供的帮助主题

 
HELP [ INDEX | topic ]

执行主机命令

 
HOST [ command ]

显示SQL * Plus系统变量或环境设置

 
SHOW { ALL | ERRORS | USER | system_variable
  [, system_variable] ...}

更改SQL * Plus系统变量或环境设置

 
SET system_variable value

启动数据库

 
STARTUP { db_options | cdb_options | upgrade_options }

db_options具有以下语法：

[FORCE] [RESTRICT] [PFILE=filename] [QUIET]
[ MOUNT [dbname] | [ OPEN [open_db_options] [dbname] ]
| NOMOUNT ]

open_db_options具有以下语法：

READ {ONLY | WRITE [RECOVER]} | RECOVER

其中cdb_options具有以下语法：

root_connection_options | pdb_connection_options

root_connection_options具有以下语法：

PLUGGABLE DATABASE pdbname [FORCE] | [RESTRICT]
[ OPEN {open_pdb_options} ]

其中pdb_connection_options具有以下语法：

[FORCE] | [RESTRICT] [ OPEN {open_pdb_options} ]

open_pdb_options具有以下语法：

READ WRITE | READ ONLY

upgrade_options具有以下语法：

[PFILE=filename] {UPGRADE | DOWNGRADE} [QUIET]

连接到数据库

 
CONNECT [{username[/password] [@connect_identifier]
         | /
         | proxy_user [ username ] [/password]
         [@connect_identifier]}
         [AS {SYSASM|SYSBACKUP|SYSDBA|SYSDG|SYSOPER
             |SYSKM}]
         [edition=value]
        ]

注意 ：以粗体显示的方括号是语法的一部分，并不表示可选性。

列出表，视图或同义词的列定义，或函数或过程的规范

 
DESCRIBE [ schema. ] object

编辑SQL缓冲区或文件的内容

 
EDIT [ filename [ .ext ] ]

获取文件并将其内容加载到SQL缓冲区中

 
GET filename [ .ext ] [ LIST | NOLLIST ]

将SQL缓冲区的内容保存到文件中

 
SAVE filename [ .ext ] [ CREATE | REPLACE | APPEND ]

列出SQL缓冲区的内容

 
LIST [ n | n m | n LAST ]

删除SQL缓冲区的内容

 
DEL [ n | n m | n LAST ]

在SQL缓冲区中的当前行之后添加新行

 
INPUT [ text ]

将文本附加到SQL缓冲区中当前行的末尾

 
APPEND text

在SQL缓冲区的当前行中查找并替换第一次出现的文本字符串

 
CHANGE sepchar old [ sepchar [ new [ sepchar ] ] ]

sepchar可以是任何非字母数字的ASCII字符，例如“/”或“！“

捕获文件中的查询结果，并可选择将文件内容发送到默认打印机

 
SPOOL [ filename[ .ext ]
        [ CREATE | REPLACE | APPEND ] | OFF | OUT ]

运行存储在文件中的SQL * Plus语句

 
@ { url | filename [ .ext ] } [ arg ... ]START { url | filename [ .ext ] } [ arg ... ]

如果文件扩展名是.sql，则可以省略ext

执行存储在SQL缓冲区中的命令

 
/

列出并执行存储在SQL缓冲区中的命令

 
RUN

执行单个PL / SQL语句或运行存储过程

 
EXECUTE statement

断开与数据库的连接

 
DISCONNECT

关闭数据库

 
SHUTDOWN
  [ ABORT | IMMEDIATE | NORMAL | TRANSACTIONAL [LOCAL] ]

退出SQL * Plus

 
{ EXIT | QUIT }
 [ SUCCESS | FAILURE | WARNING | n | variable | :BindVariable ]
