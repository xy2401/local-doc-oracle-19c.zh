11.4在Oracle Real Application Clusters（Oracle RAC）部署中使用Memory Guard

Memory Guard在打开时自动检测和监控Oracle Real Application Clusters（Oracle RAC）或Oracle RAC One Node数据库。

当Memory Guard检测到数据库节点上的内存压力时，Memory Guard会发送警报通知。您可以在$ORACLE_BASE/crsdata/ node name /qos/logs/dbwlm/auditing审计日志中找到Memory Guard警报。
