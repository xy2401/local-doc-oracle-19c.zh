2.25主持人

目的

使用HOST命令从RMAN中调用操作系统命令行子shell。

句法

host :: =


插图描述host.eps

先决条件

在RMAN提示符下或在RUN命令的大括号内执行此命令。

语义

语法元素 描述

主办

显示命令提示符并在退出子shell后继续（请参阅示例2-104 ）。

    ' command '

在指定的字符串中运行该命令，然后继续（参见示例2-105 ）。

例子

示例2-104托管备份中的操作系统

此示例生成数据文件3的映像副本，主机输出到Linux提示符以检查副本是否在目录中（Linux会话输出缩进并以粗体显示），然后恢复RMAN会话：

示例2-105在RMAN中执行操作系统副本

此示例备份数据文件system01.dbf ，然后执行Linux ls命令以显示/disk2目录中的所有文件：

BACKUP DATAFILE '?/oradata/prod/system01.dbf'
  FORMAT '/disk2/system01.dbf';
HOST 'ls -lt /disk2/*';
