运行Oracle ORAchk升级准备评估

在升级Oracle Grid Infrastructure之前，请下载并运行Oracle ORAchk升级准备情况评估。

Oracle ORAchk是Oracle RAC配置审计工具。Oracle ORAchk升级准备评估可用于获取针对Oracle Grid Infrastructure 11.2.0.3,11.2.0.4,12.1.0.1,12.1.0.2,12.2,18c和19c升级的自动升级特定运行状况检查。您可以运行Oracle ORAchk升级准备评估工具，并自动执行许多手动升级前和升级后检查。

Oracle建议您从My Oracle Support下载并运行最新版本的Oracle ORAchk。有关下载，配置和运行Oracle ORAchk的信息，请参阅My Oracle Support说明1457357.1。
