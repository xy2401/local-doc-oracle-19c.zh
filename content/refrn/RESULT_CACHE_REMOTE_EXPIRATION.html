1.294 RESULT_CACHE_REMOTE_EXPIRATION

RESULT_CACHE_REMOTE_EXPIRATION指定允许使用远程对象的结果保持有效的分钟数。

属性 描述

参数类型

整数

默认值

0

可修改

ALTER SESSION ， ALTER SYSTEM

可在PDB中修改

是

价值范围

0到操作系统相关

基本

没有

将此参数设置为0意味着不应缓存使用远程对象的结果。将此参数设置为非零值可能会产生过时的答案（例如，如果在远程数据库中修改了结果使用的远程表）。
