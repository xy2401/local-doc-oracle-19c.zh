1.253 PARALLEL_MIN_TIME_THRESHOLD

PARALLEL_MIN_TIME_THRESHOLD指定语句在考虑自动并行度之前应具有的最短执行时间。

属性 描述

参数类型

串

句法

PARALLEL_MIN_TIME_THRESHOLD = { AUTO | integer }

默认值

汽车

可修改

ALTER SESSION ， ALTER SYSTEM

可在PDB中修改

是

基本

没有

默认情况下，此参数设置为10秒。仅当PARALLEL_DEGREE_POLICY设置为ADAPTIVE ， AUTO或LIMITED ，才会启用自动并行度。

如果SQL语句引用的所有表都使用内存中列存储（IM列存储），则PARALLEL_MIN_TIME_THRESHOLD默认为1。
