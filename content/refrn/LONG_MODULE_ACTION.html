1.184 LONG_MODULE_ACTION

LONG_MODULE_ACTION允许对模块和操作使用更长的长度。

Oracle RAC

必须在所有实例上使用相同的值。

在Oracle Database 12 c Release 2（12.2.0.1）之前的Oracle Database发行版中，模块长度为48个字节，操作长度为32个字节。

从Oracle Database 12 c第2版（12.2.0.1）开始：

  • 如果LONG_MODULE_ACTION设置为TRUE （默认值），则模块和操作的长度各为64字节。

  • 如果LONG_MODULE_ACTION设置为FALSE ，则模块长度为48个字节，动作长度为32个字节。
