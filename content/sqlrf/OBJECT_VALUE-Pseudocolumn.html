OBJECT_VALUE伪列

OBJECT_VALUE伪OBJECT_VALUE对象表， XMLType表，对象视图或XMLType视图的列返回系统生成的名称。此伪列对于标识对象表中可替换行的值以及WITH OBJECT IDENTIFIER子句创建对象视图非常有用。

注意：

在早期版本中，此伪列称为SYS_NC_ROWINFO$ 。该名称仍然支持向后兼容性。但是，Oracle建议您使用更直观的名称OBJECT_VALUE 。
